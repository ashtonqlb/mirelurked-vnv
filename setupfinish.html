Manual Downloads

Root Mods

The following mods, unlike most others, need to be installed directly into the game's Root folder, not through Mod Organizer 2

Use the Manual Download and follow the instructions provided for each mod.

Universal MPI Phraser

The following mod only comes with a Windows installer by default.

Due to licensing restrictions, You must download a Linux installer for this mod manually. Sorry.

Installation instructions

  1. Extract the archive to your Downloads folder
  2. Take note of the executable named ttw_linux_gui. This is what we will use to install mods packaged as .mpi files

FNV BSA Decompressor

Decompresses the vanilla BSA files to reduce loading times and stuttering. Can also fix certain sound effects not playing.

Installation instructions

  • Main Files - FNV BSA Decompressor
    1. Extract the archive to your Downloads folder.
    2. Move the file FNV BSA Decompressor.mpi to your Universal MPI Phraser folder
    3. Open the Universal MPI Phraser. If the FNV BSA Decompressor MPI file is not automatically selected, click Browse to select it.
    4. In the box marked Fallout New Vegas select your Root folder.
    5. In the box marked Decompressed Archives, select your Data folder.
    6. Click Install MPI.
    7. Wait for the process to finish. Once it has finished, you may exit the program.

Ultimate Edition ESM Fixes Remastered for Linux

Optimizes and fixes thousands of records across the base game and DLC plugins to increase performance and stability.

Installation instructions

  • Main Files - Ultimate Edition ESM Fixes Remastered for Linux
    1. Open Dolphin
    2. Navigate to the folder where you installed Mod Organizer 2
    3. Create an empty folder called Fixed ESMs inside of the Mods folder.
    4. Take note of this new folder's path. If you installed MO2 in the default folder it will be ~/modorganizer2-newvegas/Viva New Vegas/mods/Fixed ESMs
    5. Extract the Ultimate Edition ESM Fixes Remastered archive to your Downloads folder.
    6. Open the Installer. If the Ultimate Edition ESM Fixes Remastered MPI file is not automatically selected, click Browse to select it.
    7. In the box marked Fallout New Vegas select your Root folder.
    8. In the box marked Mod folder, click Browse and navigate to the 'Fixed ESMs' folder you created in Step 3.
    9. Click Install MPI.
    10. Wait for the process to finish. Once it has finished, you may exit the program.

4GB Patcher

Main Files - FNV4GB for Linux
  1. Open the downloaded archive and move the executable file to the game's Root folder.
  2. Double-click on FalloutNVPatcher to run it.
  3. A Konsole window will open and it should say:
    FalloutNV.exe patched!
  4. Close Konsole and a file named FalloutNV_backup.exe should appear in the game's Root folder.

Finish

Installing Viva New Vegas

You have now finished the missing Linux setup section of Viva New Vegas. You can proceed with the rest of the guide, including VNV Extended, but be aware that you only need to follow Performance Mods section of the Performance and Stability Guide. You may optionally follow the guidance of the FPS Limits section.

Once you're done, make sure it's fully working; fire up a new save and walk out of Doc's shack. Make sure the Tweaks menu has appeared in the game settings and the pause menu is listing the version numbers of various installed mods at the bottom left.

Why skip most of the Performance and Stability guide?

Most (not quite all) of that section of the guide is concerned with making Fallout use a more modern rendering pipeline. SteamOS already ships with a modern rendering pipeline, and it's built in Game Mode session comes with pretty much everything you would have to do by hand (Including the frame limiter!).

That all said, performance mods are performance mods and will improve your framerate/game stability. So that bit, you should absolutely pay attention to.
